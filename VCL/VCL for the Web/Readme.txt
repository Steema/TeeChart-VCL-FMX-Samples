Please install the appropiate package:
--------------------------------------

NOTE: If your Delphi and Intraweb versions do not match
the ones included, you can make a copy of the packages
and manually edit them to refer to the appropiate version
numbers.

-------------------------------------------------------------------
For Delphi / C++ 2007 (RAD Studio 5.0)and TeeChart Pro 2012 and Intraweb 9:

-Compile IW90TeeChartPro9_110 and DCLIW90TeeChartPro9_110 packages.

-Install DCLIW90TeeChartPro9_110.dpk package


-------------------------------------------------------------------
For Delphi 7 and TeeChart Pro 2012 and Intraweb 9:

-Compile IW90TeeChartPro9_70 and DCLIW90TeeChartPro9_70 packages.

-Install DCLIW90TeeChartPro9_70.dpk package


-------------------------------------------------------------------
For Delphi 6 and TeeChart Pro 2012 and Intraweb 9:

-Compile IW70TeeChartPro9_60 and DCLIW70TeeChartPro9_60 packages.

-Install DCLIW70TeeChartPro9_60.dpk package


==========
Important:
==========

The included packages and demo have been created
using Intraweb versions 7.0.11, 9.0.16, 10 and TeeChart Pro 2010.01

If you are using a different Intraweb version, you 
might need to modify the packages (replace Intraweb
package names in the "requires" section).
