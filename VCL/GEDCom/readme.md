## TeeChart and GEDCom genealogical data

This example shows TeeChart examples using genealogical data from a GEDCom file.

A [GEDCom file](https://www.gedcom.org/) is a standard format used by many webs and software aplications like:

- FamilySearch.org
- Ancestry
- MyHeritage
- GeneaNet
- GEDMatch
- and many others

A Delphi unit is included to read and parse a *.ged file to create an internal typed structure.

### Example using the "United States Presidents tree"

A genealogical tree of 2322 persons (families of the Presidents of the United States), found here:
